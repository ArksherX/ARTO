================================================================================

VerityFlux 2.0 is the first open-source AI security framework with complete OWASP 
LLM Top 10 (2025) and Agentic AI Top 10 (2026) coverage. As autonomous agents 
move into production with access to tools, credentials, and decision-making 
authority, traditional security fails to address runtime threats like goal 
hijacking, tool misuse, and memory poisoning.

This interactive demo showcases the complete framework: scan your own LLM or 
agent in 30 seconds, watch the Cognitive Firewall block goal hijacking attacks 
in real-time, and explore Flight Recorder's compliance logging (GDPR, SOC 2, 
ISO 27001 ready). The demo includes multiple interfaces—Streamlit web UI for 
interactive testing, Flask REST API for CI/CD integration, and CLI for automation.

Attendees can bring their own agents to scan, try to bypass the firewall, and 
contribute test cases via GitHub. VerityFlux reduces goal hijacking failure 
rates from 50% to <10% through semantic drift detection and provides complete 
audit trails for compliance. Used in production healthcare and fintech 
deployments with demonstrated 40% attack surface reduction.

All code, documentation, and integration guides provided. Come hack on AI 
security with the only framework covering the entire 2025/2026 OWASP threat
